Розрахунково-графічна робота (РГР)
Довженко Л.П. КІ-222

Тема: Автоматизація міграцій NoSQL бази даних MongoDB за допомогою Liquibase та візуалізація даних на Node.js.

Цей репозиторій містить результати виконання РГР з налаштування Liquibase для керування схемою даних у MongoDB, а також вебзастосунок для відображення результатів міграції.

Структура проєкту:
db/changelog/changelog.xml — опис структури даних та початкових записів користувачів (Коваленко, Петренко, Довженко тощо).

lib/ — драйвери MongoDB та розширення Liquibase, необхідні для роботи з NoSQL.

public/ — клієнтська частина сайту (HTML-картки та CSS-стилі) з унікальним дизайном.

server.js — бекенд на Node.js (Express), що реалізує API для отримання даних з MongoDB.

liquibase.properties — параметри підключення до mongodb://localhost:27017/rgr_db.

Робота з проєктом:
1. Міграція бази даних
Для завантаження даних у MongoDB виконайте:

PowerShell
liquibase update
(Примітка: Переконайтеся, що драйвери в папці lib доступні в classpath).

2. Запуск веб-інтерфейсу
Для перегляду результатів у браузері:

Встановіть залежності: npm install.

Запустіть сервер: node server.js.

Відкрийте: http://localhost:3000.

Результат:
Після виконання міграції дані відображаються у вигляді інтерактивних карток профілів, що підтверджує успішну синхронізацію структури бази даних та програмного коду.
